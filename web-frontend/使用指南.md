# 使用指南

## 快速开始

### 1. 启动服务

**启动后端：**
```bash
cd web-frontend/server
pip install -r requirements.txt
python gtp_bridge.py
```

**启动前端：**
```bash
cd web-frontend
npm install
npm run dev
```

### 2. 打开浏览器

访问 `http://localhost:3000`

## 功能说明

### 连接AI

1. 选择AI模式：
   - **策略网络**：快速响应，适合实时对弈
   - **MCTS搜索**：更强棋力，但计算较慢
2. 点击"连接AI"按钮
3. 等待连接成功（状态显示"已连接"）

### 游戏模式

#### 手动模式
- 可以自由点击棋盘落子
- 需要手动点击"AI走子"按钮让AI下棋
- 适合分析、复盘等场景

#### 人机对弈
- 自动交替下棋
- 你落子后，AI会自动回应
- 适合与AI对弈练习

#### AI自动对弈
- 两个AI自动对弈
- 可以观察AI的下棋风格
- 适合测试AI性能

### 操作说明

- **落子**：直接点击棋盘上的交叉点
- **清空棋盘**：点击"清空棋盘"按钮重新开始
- **AI走子**：在手动模式下，点击此按钮让AI下棋

### 棋盘说明

- **黑色棋子**：黑方（通常先手）
- **白色棋子**：白方
- **红色标记**：最后一步落子位置
- **坐标**：棋盘边缘显示坐标（字母和数字）

## 注意事项

1. **模型文件**：确保已训练好所有模型（policyNet.pt, playoutNet.pt, valueNet.pt）
2. **合法走子**：只能在没有棋子的交叉点落子
3. **打劫规则**：AI会自动处理打劫规则
4. **网络延迟**：MCTS模式可能需要几秒时间思考

## 常见问题

### Q: 点击连接AI无反应？

A: 检查：
- 后端服务是否已启动
- 模型文件是否存在
- 浏览器控制台是否有错误信息

### Q: 落子无效？

A: 确保：
- 已成功连接AI
- 轮到你的回合（查看"当前玩家"）
- 点击的位置是空的

### Q: AI走子很慢？

A: MCTS模式需要较长时间计算，这是正常的。可以：
- 切换到"策略网络"模式获得更快响应
- 等待AI完成思考（显示"AI思考中..."）

### Q: 棋盘显示不正确？

A: 尝试：
- 点击"清空棋盘"重新开始
- 刷新页面重新连接
