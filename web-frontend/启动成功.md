## 服务管理

### 使用启动脚本（推荐）

**Windows批处理：**
```bash
cd web-frontend
start.bat
```

**PowerShell：**
```powershell
cd web-frontend
.\start.ps1
```

### 手动启动

**终端1 - 后端服务：**
```bash
cd web-frontend/server
python gtp_bridge.py
```

**终端2 - 前端服务：**
```bash
cd web-frontend
npm run dev
```

## 功能说明

1. **连接AI**: 在控制面板选择AI模式，点击"连接AI"
2. **开始对弈**: 选择游戏模式，点击棋盘落子
3. **清空棋盘**: 点击"清空棋盘"重新开始

## 注意事项

- 确保已训练好所有模型文件（policyNet.pt, playoutNet.pt, valueNet.pt）
- 如果遇到连接问题，检查后端服务是否正常运行
- 查看浏览器控制台获取详细错误信息

## 停止服务

- 关闭对应的命令行窗口
- 或使用 Ctrl+C 停止服务
